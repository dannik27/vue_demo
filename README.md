# vue_demo

## Project setup
```
npm install
```

```
cd back
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles to build folder
```
npm run build
```

### Compiles to backend public folder
```
npm run build-express
```

### Run backend
```
node back/server.js
```
